# ポケモンタイプ相性チェッカー

防御側ポケモンのタイプ（最大2つ）と特性を選ぶと、攻撃側全18タイプとの相性倍率を一目で確認できるWebアプリです。HTML/CSS/JavaScriptを1ファイルに収めた自己完結構成で、サーバも外部ライブラリも使いません。

**▶ 今すぐ使う: https://sgkz0920.github.io/pokemon-type-chart/**

## 主な機能

- **タイプ選択**: 18タイプから最大2つ選択（複合タイプ対応）。再クリックで解除、3つ目を選ぶと古い方が自動で外れます
- **相性計算**: 第6世代以降のタイプ相性表に準拠（はがねに対するあく・ゴースト等倍など）
- **特性補正**: 「ふゆう」「あついしぼう」「フィルター」「ふしぎなまもり」など、技のタイプに作用する防御側特性16種に対応
- **倍率別表示**: 計算結果を倍率ごとにグルーピングし、タイプごとのイメージカラーで表示。特性補正で生じる3倍・1.5倍・0.625倍などの中間倍率にも対応
- **レスポンシブUI**: PC・スマートフォンどちらでも操作しやすいレイアウト

## 使い方

### ブラウザで使う
上記URLを開き、防御側のタイプ（と必要なら特性）をタップするだけです。

### iPhoneでアプリのように使う
1. Safariで https://sgkz0920.github.io/pokemon-type-chart/ を開く
2. 共有ボタン（□↑）→「**ホーム画面に追加**」
3. ホーム画面のアイコンからワンタップで起動できます

### オフラインで使う
[src/pokemon_type_chart.html](src/pokemon_type_chart.html) をダウンロードしてブラウザで開くだけで、ネットワークなしで全機能が動作します。

## 技術構成

- **HTML / CSS / JavaScript（Vanilla）** — フレームワーク・ビルド工程なし、単一ファイル
- **アーキテクチャ**: データ層（タイプ定義・18×18相性表・特性ルール）／ロジック層（純粋関数による倍率計算・特性補正）／表示層 の3層構成。状態変更のたびに全再描画する単方向データフロー
- **特性の表現**: 効果を `immune`（無効化）/ `scale`（定率補正）/ `scaleIfSuper`（効果ばつぐん時のみ補正）/ `wonderGuard` の4種の宣言的ルールで定義しており、特性の追加はデータ1行で完結
- **ホスティング**: GitHub Pages（`main` ブランチから配信、Jekyll無効）

設計の詳細は [doc/](doc/) 配下の各設計書を参照してください。

## 開発

- 機能追加は設計書の作成・更新 → ブランチ上で実装 → 検証 → `main` へマージ、の流れで進めています
- 動作検証はPlaywright（ヘッドレスEdge）でブラウザを実操作するスクリプトにより、タイプ選択・特性補正・表示グルーピングを確認しています
- `main` へのpushで公開サイトに自動反映されます（反映まで1〜2分）
