# ポケモンタイプ相性チェッカー 追加設計書（ポケモン特性）

- 作成日: 2026-07-04
- 対象バージョン: ver2（Electron版）
- 前提設計書: [pokemon_type_chart_pokemon_search_design.md](./pokemon_type_chart_pokemon_search_design.md)

## 1. 目的・要件

| No | 要件 |
|----|------|
| F-19 | data/pokemon.json に、各ポケモンが備えている可能性のある特性情報を追加する |
| F-20 | ポケモン選択時（検索候補から選択）、特性セレクトをそのポケモンが持つ可能性のある特性のみに絞り込む |
| F-21 | 可能性のある特性が1つのみの場合、「特性なし」は表示しない。その特性のみ選択状態にする |
| F-22 | タイプ選択に戻った場合は、全特性（＋「特性なし」）から選べるようにする |

## 2. データ設計（F-19）

### 2.1 data/pokemon.json の拡張

```json
{
  "_meta": { ... },
  "pokemon": [
    {
      "no": 1,
      "name": "フシギダネ",
      "types": [4, 7],
      "abilityIds": [0]  // abilities.json のID。ただし隠れ特性は後方互換のため対象外（将来拡張）
    },
    {
      "no": 6,
      "name": "リザードン",
      "types": [1, 9],
      "abilityIds": [1, 3]  // 炎, 1種以上複数の可能性あり
    }
  ]
}
```

- `abilityIds`: 配列。各ポケモンが持つ可能性のある特性の ID（0〜16）。
  通常特性1〜2個。隠れ特性は対象外（データ取得が別）。

### 2.2 PokeAPI からの取得

生成スクリプト（scripts/generate-pokemon-json.mjs）で、各ポケモンの`abilities`（通常特性のみ）を PokeAPI から取得し、タイプと同時に保存する。

## 3. UI設計（F-20 〜 F-22）

特性セレクトの動作を変更する。

| 場面 | 表示内容 | 挙動 |
|------|---------|------|
| 初期状態 ／ タイプ選択 | 全17種（特性なし + 16の特性） | リセット可能、全特性から選択 |
| ポケモン検索で選択 | そのポケモンが持つ特性のみ | リセット不可。1つのみの場合は表示が変わるが、選択は固定 |
| タイプ選択に戻す | 全17種に戻す | リセット可能 |

**具体例**:
- フシギダネ（くさ×どく）を選択 → 特性セレクト = 「しんりょく」のみ。「特性なし」は非表示。
  ジムリーダーなど1特性固定のポケモンも同じく。
- リザードン（ほのお×ひこう）を選択 → 特性セレクト = 「もうか」「サンパワー」の2択。デフォルトは「もうか」。
- クリア時や「タイプで選択」に戻す → 特性セレクト = 「特性なし」を含む全17種に復帰。

## 4. 実装設計

| ファイル | 変更内容 |
|----------|---------|
| scripts/generate-pokemon-json.mjs | PokeAPI から各ポケモンの通常特性を取得し、`abilityIds` を追加 |
| data/pokemon.json | 生成済み。各ポケモンに `abilityIds` を追加 |
| src/renderer/app.js | ポケモン選択時に特性セレクトを動的に更新する `updateAbilityOptions(abilityIds)` を追加 |
| src/renderer/index.html | 特性セレクトを空にしないため、初期 `<option>` は削除（app.js で生成） |

## 5. テスト計画

### 5.1 単体テスト（追加）

| No | 検証内容 |
|----|----------|
| U10 | pokemon.json の各件について `abilityIds` が1〜2要素の配列で、値が0〜16の範囲内であること。代表値としてフシギダネ=[0]、リザードン=[1,3]、ピカチュウ=[2] など |

### 5.2 E2E検証（追加）

| No | 検証内容 |
|----|----------|
| E13 | ポケモン「ピカチュウ」（特性1つ）を選択すると、特性セレクトが「ちくでん」のみになり、「特性なし」は表示されない |
| E14 | ポケモン「リザードン」（特性2つ）を選択すると、特性セレクトが「もうか」「サンパワー」の2択（「特性なし」非表示）。「タイプで選択」に戻すと全特性が復帰する |
