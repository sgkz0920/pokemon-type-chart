# ポケモンタイプ相性検索ツール 仕様書

## 1. 概要
本ツールは、ポケモンの防御側のタイプ（単一または複合）を指定することで、攻撃側の全18タイプとの相性（ダメージ倍率）を直感的に確認できるWebアプリケーションです。

## 2. 機能仕様

### 2.1 タイプ選択機能
- **防御側タイプ設定**: 
    - 18種類のタイプボタンから、最大2つまで選択可能です。
    - 1つ選択された状態で同じタイプを再度クリックすると選択解除されます。
    - 3つ目を選択しようとした場合、古い選択を解除して新しく選択されたタイプをセットします。

### 2.2 相性計算ロジック
- **倍率算出**: 
    - 攻撃側の各タイプに対し、防御側のタイプ1とタイプ2それぞれの相性を掛け合わせます。
    - 算出される倍率: 4倍、2倍、1倍、0.5倍、0.25倍、0倍。
- **最新仕様への準拠**: 
    - 第6世代以降のタイプ相性表（鋼タイプへの悪・ゴースト等倍など）を基準とします。

### 2.3 表示機能
- **結果のリアルタイム更新**: 
    - ユーザーがタイプボタンをクリックするたびに、即座に計算結果を再描画します。
- **倍率別グルーピング表示**: 
    - 倍率ごとにセクションを分け（例：「4倍」「2倍」「耐性あり」など）、どのタイプが有効かを一目で判別できるようにします。
    - 各タイプ名は、そのタイプを象徴するイメージカラーで色付けします。

### 2.4 GUI構成
- **メインウィンドウ**:
    - **タイプ選択パネル（上部）**: 18タイプのボタンをタイル状に配置。
    - **選択ステータス（中央）**: 現在選択されているタイプ（1つ目、2つ目）を大きく表示。
    - **結果表示パネル（下部）**: 計算された倍率リストを表示。
    - **クリアボタン**: 選択状態をすべてリセットする機能。

## 3. 技術スタック
- GoogleDriveに格納したHTML上の操作で完結すること
- UIはモダンな形にすること

## 4. 拡張機能（将来的な検討）
- **特性の考慮**: 「ふゆう」や「あついしぼう」などの特性を選択肢に加え、より正確なダメージ計算を可能にする。
- **逆引き機能**: 攻撃側のタイプを選択し、どのタイプに対して有利・不利かを表示するモードの追加。
